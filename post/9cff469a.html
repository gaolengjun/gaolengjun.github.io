【10.23更新】Windows 11 24H2 26100.2161 X64 最新中文正式版

发表于2024-10-26|更新于2024-10-26|软件分享
|阅读量:


微软向Windows11 24H2用户推出了十月非安全可选更新 KB5044384 ，内部版本号为26100.2161，此次更新修复了诸多 BUG，包括 8.63GB 无法删除的升级数据、DirectAccess、SFC scannow 错误等问题。系统采用微软官方 Windows 11 24H2 26100.2161 专业版进行优化，安全无毒，保留 Edge 浏览器，完美支持新老机型，安装好后系统分区占用 12.7 GB。

Win11 24H2 版本更新的优势

　　最新的 Windows 11 2024 更新旨在通过引入 Wi-Fi 7 来增强连接性，通过新的快速设置提高工作效率，以及通过高级辅助听觉设备支持改进辅助功能。 此外，它还引入了节能功能，可延长电池使用时间并降低能耗。
　　1. Wi-Fi7支持：提升连接体验。
　　2. 辅助助听设备支持的蓝牙LE音频增强功能。
　　3. 系统托班&任务栏功能增强功能。
　　4. 更简化文件资源管理器。
　　5. 适用于电脑的智能电源管理。
　　6. 使用 QR 码加入和共享Wi-Fi网络。
　　7. 用于 Wi-Fi 网络访问的增强隐私控制。
　　8. Microsoft TEAMS 中的轻松账户管理和通知。
　　9. 跨设备扩展语音Clarity。

系统介绍

　　1. 系统已集成最新补丁至2024.10，此版本为 Windows11 24H2 版本正式版，会逐步向用户推送更新。
　　2. 系统离线制作，确保安全无毒，优化方案全新升级，使得系统运行更加流畅。
　　3. 默认Administrator账户登录。
　　4. 自动跳过硬件检测，去除不满足安装条件，老爷机也能安装
　　5. 默认状态下系统自动更新功能是关闭的，但可以支持更新补丁，用户可以根据需要手动开启。
　　6. 系统经过严格测试，包括多款游戏和应用的兼容性测试，确保稳定。
　　7. 集成万能主板驱动与网卡版，并使用驱动智能识别技术，大大减少驱动误装蓝屏几率。
　　8. 安装过程智能化，无需专业知识，无需人工干预，一键硬盘安装。
　　9. 更新完成后，系统将自动进行硬盘扫描，确保没有木马病毒和垃圾文件，保障用户使用安全。

系统已安装更新

　　1. 系统已安装用于 Microsoft Windows 的 更新(KB5042098)。
　　2. 系统已安装用于 Microsoft Windows 的 安全更新(KB5044384)。

更新日志

一. 逐步推出

　　1.1、通知
　　用户可以停止某些应用程序的通知建议。用户选择通知中的省略号 (…) 并将其关闭，也可以转到设置 > 系统 > 通知来关闭它。在发件人列表中查找“通知建议”。只有在您收到通知后，条目才会出现在列表中。
　　1.2、Wi-Fi 密码对话框
　　该对话框现在采用 Windows 11 的视觉设计，用户打开设置 > 网络和互联网后可以看到。
　　1.3、讲述人
　　微软增加了一个新快捷键“讲述人键 + Control + X”，用于将讲述者最后说过的话复制到剪贴板。这个快捷键遵循“讲述者键 + X”的模式，它会大声重复最后说过的短语，这是记忆类似快捷键的好方法。
　　讲述人现在将自动读取新版 Outlook 应用中的邮件内容，类似于 Outlook 经典版的行为。
　　1.4、Gamepad 键盘
　　本次更新推出全新的游戏手柄键盘布局，用户可以使用 Xbox 游戏手柄在屏幕上移动并输入。
　　1.5、开始菜单
　　开始菜单上“All apps”现更名为“All”。
　　1.6、ALT + Tab
　　修复在某些 PC 上，当用户在特定窗口之间切换时，屏幕会黑屏几秒钟。
　　1.7、Scanning apps
　　修复无法检测到某些已连接的扫描仪。

二. 改进

　　2.1、虚拟硬盘(VHD / VHDx)
　　此更新在设置中添加了一个取消挂载虚拟硬盘按钮。在您的 VHD 或 VHDx 的属性中，转到设置 > 系统 > 存储 > 磁盘和卷。
　　2.2、网络连接
　　用户使用 netsh wlan show networks 命令，应该能够读取 UTF-8 编码的 SSID。这意味着包含 Unicode 字符(如表情符号)的 Wi-Fi SSID 应在 netsh 输出中正确显示。
　　2.3、UI
　　修复：在运行某些应用时，您无法查看 UI 的某些部分。
　　2.4、实时内核调试文件(转储)
　　修复 win32kbase.sys 在不应该创建实时内核调试文件问题。
　　2.5、Winlogon
　　修复关机时停止错误。
　　2.6、Windows Kernel Vulnerable Driver Blocklist 文件 DriverSiPolicy.p7b
　　此更新将易受“自带易受攻击驱动程序”(BYOVD) 攻击的驱动程序添加到列表中。
　　2.7、网页登录
　　修复屏幕停止响应，无法通过网页登录您的账户问题。
　　2.8、sfc /scannow 命令
　　修复该命令运行错误。
　　2.9、HTML 应用程序(HTA)
　　HTA 可选组件 (OC) 现在可以在 ARM64 Windows PE 中使用。

三. 修复

　　3.1、网页登录
　　修复无法通过网页登录账号问题。
　　3.2、重映射 Copilot 按键
　　微软正在增加配置 Copilot 按键的功能。用户可以选择让 Copilot 按键启动经过 MSIX 封装和签名的应用程序(该应用程序符合安全和隐私要求，以确保客户安全)。在已安装 Copilot 应用程序的设备上，Copilot 按键默认启动 Copilot，客户可以选择不同的体验。
　　3.3、Windows 磁盘清理应用
　　此更新解决了错误的可用空间估算的一些原因，并提高了其准确性。

系统配置要求

　　1. 处理器：1 GHz 64位处理器。
　　2. 内存：4 GB及以上。
　　3. 显卡：带有 WDDM 1.0或更高版本驱动程序Direct×9图形设备。
　　4. 硬盘可用空间：30 G以上（主分区，NTFS格式）。
　　5. 显示器：要求分辨率在1024×768像素及以上，或可支持触摸技术的显示设备。
